●關於範例檔案
　範例檔案包含於《TensorFlow開發入門　以Keras建置深度學習模型手法》內文解說的第3章至第12章的範例。

●本書的範例測試環境與支援GPU的狀態
　本書的範例檔案可於下列的環境正常執行。
　此外，要於電腦執行第2部的範例檔案，必須搭載GPU（規格等同於NVIDIA公司的GeForce GTX1080Ti）。未搭載GPU的電腦不建議執行第2部的範例檔案，以免造成CPU過重的負荷。

・第1部
OS：Windows 10
CPU：Intel Core i5 3.00GHz、4核心
記憶體：8GB
GPU：無
Python：3.5.4/3.5.5
Anaconda：5.0.1
TensorFlow：1.5.0

・第2部（GPU的規格必須等同於NVIDIA公司的GeForce GTX1080Ti）
OS：Ubuntu 16.04.4 LTS
CPU：Intel Xeon E5-1650 v4 3.60GH、6核心
記憶體：64GB
GPU：GeForce GTX1080Ti
Python：3.5.4/3.5.5
TensorFlow-GPU：1.5.0
